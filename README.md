# StackWise

AI Spend Optimization Platform for startups and teams.

StackWise audits AI tool subscriptions and identifies opportunities to reduce unnecessary spend through recommendation-based analysis.

## Features

- AI tool usage audit
- Optimization scoring
- Estimated monthly waste calculation
- Annual savings projection
- Smart recommendations
- Responsive SaaS-style UI

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React

## Current Functionality

Users can:
- Enter team size
- Add monthly AI spend
- Add tools currently used
- Select API usage level
- Generate optimization recommendations

## Future Improvements

- Dynamic pricing database
- Smarter recommendation engine
- Authentication
- Saved audit history
- Charts and analytics dashboard
- AI-generated optimization suggestions
