# STACKWISE PRODUCT FLOW

## Goal
Help startups reduce unnecessary AI subscription and API spending.

---

## User Flow

1. User opens landing page

2. User clicks "Start Free Audit"

3. User fills audit form:
   - Team size
   - AI tools used
   - Monthly AI spend
   - API providers used
   - Duplicate tools

4. System analyzes:
   - overlapping tools
   - excessive subscriptions
   - cheaper alternatives
   - estimated waste

5. System generates:
   - optimization score
   - monthly savings
   - annual savings
   - recommendations

6. User sees audit results dashboard

7. User can share or save results
